# ☁️ 雲端同步設定教學（一次性，約 10 分鐘）

做完這份，你就能「手機記、電腦開同一網站登入同一個 Google 帳號 → 紀錄自動同步」。
全程免費。請用**電腦的瀏覽器**操作比較好點。

> 你只要完成下面 7 步，最後把**第 3 步的那段設定碼**貼給我，我幫你裝上去就完成 ✅

---

## 第 1 步：建立 Firebase 專案

1. 開 👉 https://console.firebase.google.com/
2. 用你的 Google 帳號（vkopkimo@gmail.com）登入。
3. 點 **「建立專案 / Create a project」**。
4. 專案名稱打：`delivery-tracker`（隨意），按繼續。
5. 「Google Analytics」這頁可以**關掉**（把開關關閉）或直接略過，按 **建立專案**。
6. 等它跑完，按 **繼續**。

## 第 2 步：新增「網頁應用程式」

1. 進到專案首頁後，找到中間的圖示列，點那個 **`</>`（網頁 Web）** 圖示。
   （文字通常是「將 Firebase 新增至您的網頁應用程式」）
2. 「應用程式暱稱」打：`外送跑單`，**不用**勾「Firebase Hosting」，按 **註冊應用程式**。

## 第 3 步：複製設定碼（⭐ 這段要傳給我）

註冊後，畫面會出現一段程式碼，長這樣：

```js
const firebaseConfig = {
  apiKey: "AIzaSy............",
  authDomain: "delivery-tracker-xxxx.firebaseapp.com",
  projectId: "delivery-tracker-xxxx",
  storageBucket: "delivery-tracker-xxxx.appspot.com",
  messagingSenderId: "123456789012",
  appId: "1:123456789012:web:abcdef123456"
};
```

👉 **把整段 `firebaseConfig = { ... }` 複製起來，貼到對話框給我就好。**
（這些值是可以公開的，貼出來很安全，不是密碼。）

按 **繼續操作主控台 / Continue to console** 回到首頁。

## 第 4 步：開啟「Google 登入」

1. 左邊選單點 **建構 / Build → Authentication（驗證）**。
2. 按 **開始使用 / Get started**。
3. 上方分頁點 **Sign-in method（登入方式）**。
4. 在清單裡點 **Google** → 把右上角開關打開 **啟用**。
5. 下方「專案的公開名稱」隨意；「支援電子郵件」選你的 Gmail。
6. 按 **儲存**。

## 第 5 步：允許我們的網站登入（重要！）

1. 一樣在 **Authentication** 裡，上方點 **Settings（設定）**。
2. 找到 **Authorized domains（已授權網域）**。
3. 按 **Add domain（新增網域）**，輸入：
   ```
   guanhung7484.github.io
   ```
   按新增。
   （`localhost` 預設就有，不用動。）

## 第 6 步：建立資料庫（Firestore）

1. 左邊選單點 **建構 / Build → Firestore Database**。
2. 按 **建立資料庫 / Create database**。
3. 位置（location）選 **asia-east1（台灣/彰化）** 或附近的亞洲區，按下一步。
4. 安全規則選 **以正式版模式啟動 / Start in production mode**，按 **啟用 / Enable**。

## 第 7 步：設定安全規則（只有你看得到自己的資料）

1. 進到 Firestore 後，上方點 **規則 / Rules** 分頁。
2. 把裡面的內容**全部刪掉**，換成下面這段：

   ```
   rules_version = '2';
   service cloud.firestore {
     match /databases/{database}/documents {
       match /delivery/{uid} {
         allow read, write: if request.auth != null && request.auth.uid == uid;
       }
     }
   }
   ```

3. 按 **發布 / Publish**。

---

## ✅ 完成！

把 **第 3 步的 `firebaseConfig` 那段**貼給我，我會：
1. 幫你填進網站、推上 GitHub。
2. 然後你打開網站，右上會出現 **「☁️ 登入 Google」**，按下去用 Gmail 登入。
3. 之後手機、電腦只要**登入同一個 Google 帳號**，紀錄就自動同步 🎉

> 同步列上的小圓點：綠色＝已同步、黃色＝儲存中、紅色＝連線有問題。
> 沒網路也能照記，等有網路會自動補傳。
